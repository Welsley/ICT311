Detailed Description

The Facade of SFML.

This class provides a Facade between some components in our engine and SFML

Author
Umar Badat
Note
Some Functions are not used in this class, while others may have been split down furthure.

Returns

Note
This function was initially used for testing and a possible way to poll for keyBoard events. The issue is that the input events are locked to the window that was created. this is a limitation by most operating systems The best way to resolve it would be to pass the window or find another method of input
